## ���️ Tech Stack

- **Backend**: Python, Flask 3.0.0, scikit-learn

- **Models**: ML classifiers with ~82% accuracy
- **Setup**: Easy setup scripts (setup.sh / setup.bat)

## ��� Quick Start

Choose any service and run:

```bash
cd [service-name]/backend
bash setup.sh          # Linux/Mac
# or
setup.bat              # Windows

python app.py
```

Backend: `http://localhost:5000`
